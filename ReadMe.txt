# news-react-app

React js

This website using https://apilist.fun/api/news as API

How to run:
1. Download this project as Zip 
2. Extract and Open folder with using Visual Studio Code
3. Go to terminal, use git bash terminal
4. in the terminal type "cd news-app"
5. "npm install
6. "npm start"

NOTE: This api has an access limit of 1000 requests and will be reset in 24 hours
